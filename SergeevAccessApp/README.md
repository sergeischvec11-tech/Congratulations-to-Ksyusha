# SergeevAccessApp

WinForms-проект для работы с базой Access `Database21_Normalized.accdb`.

## Что сделано

- Исходная таблица `[Order]` сохранена в базе для сравнения.
- Данные нормализованы в таблицы:
  - `Categories` - справочник категорий.
  - `Products` - карточки товаров, артикулы, наименования и цены.
  - `Stock` - складские остатки по товарам.
- Форма подключается к Access через `Microsoft.ACE.OLEDB.12.0`.
- Кнопка `Все товары` выводит все товары с категорией, количеством и ценой без НДС.
- Кнопка `Аксессуары на складе` выводит только аксессуары, которые хранятся на складе, с количеством и ценой без НДС.

## Как открыть

Откройте `SergeevAccessApp.sln` в Visual Studio и запустите проект. Проект собирается под `x86`, потому что на этом компьютере установлен 32-битный Access OLEDB-провайдер.
